# Sai Varshith Pujala - Developer Portfolio

This is a personal portfolio website for Pujala Sai Varshith, built with Next.js (App Router) and styled using Tailwind CSS.
It showcases his professional summary, education, technical skills, projects, certifications, and contact information.

## Features

*   **Professional Design:** Clean, modern, and professional user interface.
*   **Mobile Responsive:** Adapts seamlessly to various screen sizes, from mobile to desktop.
*   **SEO Friendly:** Optimized for search engines with appropriate metadata.
*   **Next.js App Router:** Utilizes the latest Next.js features for routing and data fetching.
*   **Tailwind CSS:** Utility-first CSS framework for rapid and consistent styling.

## Project Structure

The project is structured using the Next.js App Router:

*   `/app/page.tsx`: Home page.
*   `/app/about/page.tsx`: About me page with detailed resume information.
*   `/app/projects/page.tsx`: Projects showcase page.
*   `/app/contact/page.tsx`: Contact information page.
*   `/components`: Reusable UI components like Navbar and Footer.

## Technologies Used

*   **Framework:** Next.js (App Router)
*   **Styling:** Tailwind CSS
*   **Language:** TypeScript, JavaScript
*   **Package Manager:** npm (or yarn)

## How to Run Locally

1.  **Clone the repository:**
    bash
    git clone [repository-url]
    cd sai-varshith-portfolio
    

2.  **Install dependencies:**
    bash
    npm install
    # or yarn install
    

3.  **Run the development server:**
    bash
    npm run dev
    # or yarn dev
    

4.  Open [http://localhost:3000](http://localhost:3000) in your browser to see the portfolio.
