# Pujala Sai Varshith's Portfolio

This is a personal portfolio website for Pujala Sai Varshith, built using Next.js (App Router) and styled with Tailwind CSS.

## Features
- Professional and clean UI
- Mobile responsive design
- SEO friendly
- Pages for Home, About, Projects, and Contact

## Setup and Run
1.  **Clone the repository:**
    bash
    git clone https://github.com/saivarshithnaidu/sai-varshith-portfolio.git
    cd sai-varshith-portfolio
    
2.  **Install dependencies:**
    bash
    npm install
    # or
    yarn install
    
3.  **Run the development server:**
    bash
    npm run dev
    # or
    yarn dev
    
    Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

4.  **Build for production:**
    bash
    npm run build
    # or
    yarn build
    
    To start the production server:
    bash
    npm run start
    # or
    yarn start
    

## Project Structure
- `app/layout.tsx`: Root layout for the application.
- `app/globals.css`: Global styles, including Tailwind CSS imports.
- `app/page.tsx`: Home page.
- `app/about/page.tsx`: About me page with education, skills, and certifications.
- `app/projects/page.tsx`: Displays projects.
- `app/contact/page.tsx`: Contact information.
- `tailwind.config.js`: Tailwind CSS configuration.
- `postcss.config.js`: PostCSS configuration.
- `package.json`: Project dependencies and scripts.
